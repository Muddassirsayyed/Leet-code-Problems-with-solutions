# 💻 Example Solution

## Two Sum

```python
def twoSum(nums, target):
    seen = {}

    for i, num in enumerate(nums):
        diff = target - num
        if diff in seen:
            return [seen[diff], i]
        seen[num] = i
```

### Complexity Analysis
- Time Complexity: **O(n)**
- Space Complexity: **O(n)**
